format new LTO tapes

If you have inserted new LTO tapes, format them here.

If there are formmated, current tapes in the drives, click here to mount them for writing.

Once the process finishes you will see a result screen showing the status and ID of each tape, or any errors that occurred.

Be patient, this takes a minute!

MOUNT LTO

After tapes are mounted, go here to choose packages to write to tape.

WRITE LTO

After tapes are mounted, go here to choose packages to get from tape.

GET DIPs

If there are tapes mounted that you need to unmount, click this button.
